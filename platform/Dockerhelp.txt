Базовые команды (Их сильно больше легче ии попросить, она вам все нужное напишет)

# Сборка и запуск
docker compose up --build

# Запуск в фоне (чтобы не занимало терминал)
docker compose up -d

# Остановка
docker compose down



Dockerfile описывать что у нас в контейнерах 

docker-compose.yml описывает взаимодействие контейнеров

У нас 2 контейнера (1 - проект, 2 - база данных)

К базе данных щас можно подключится извне, в готовой версии можно будет сделать так
Чтобы можно было только из контейнера проекта заходить

Чтобы посмотреть внутренности контейнера скачайте dbeaver, 
В нем укажите что хотите подключить посгрескл таблицу, 
При подключении указываете то что просят (данные берете из .env)
Контейнер с бд соединяется по хосту db, НО нам надо будет писать localhost
